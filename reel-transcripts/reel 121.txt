So here's how it works. Layer one are hard limits. Fixed number of requests per time window. Hit the wall, get a 429. This protects you from abuse, but it does not create a good user experience. Layer two is adaptive limits. Instead of a fixed wall, the limits adjust based on the system's health. When the server is healthy, limits are generous.
