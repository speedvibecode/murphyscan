# Reel 21 - Own the AI-Written Code

## Summary
Shipping AI-generated code without understanding it creates future tech debt. The reel recommends reading key files, renaming vague generated functions, and deleting unused code so the builder truly owns the system.

## Verification Checklist
- Inspect core routes, auth middleware, and database queries for unexplained code.
- Rename generic functions and variables to reflect real app behavior.
- Remove unused imports, functions, helpers, and unrendered components.

## Skill Signal
Use this as a maintainability scan for AI-generated codebases.
