# Reel 03 - Dependency Hygiene

## Summary
Modern apps inherit hundreds of transitive dependencies, and some will have known vulnerabilities. The reel emphasizes running audits, committing lockfiles, and treating dependency review as recurring maintenance.

## Verification Checklist
- Run the package manager audit and record severity counts.
- Confirm the lockfile is committed and not ignored or deleted.
- Check for a recurring dependency audit/update process, especially for critical CVEs.

## Skill Signal
Use this as a dependency scan: audit packages, verify lockfile discipline, and flag unowned update risk.
