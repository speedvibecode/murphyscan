# Reel 43 - Pricing Metric and Usage Events

## Summary
Pricing should track the value customers receive. The reel recommends choosing a value-aligned metric, using credits to abstract action costs, and recording every billable action as an event.

## Verification Checklist
- Match pricing to value: users, processed units, generated output, or another measurable success proxy.
- Use a credit system if direct usage pricing would expose too much complexity.
- Store billable events in a dedicated table so billing and analytics share one source of truth.
