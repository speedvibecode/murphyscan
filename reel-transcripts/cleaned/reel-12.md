# Reel 12 - Database Is Not a Spreadsheet

## Summary
AI-generated databases often become oversized tables with no relationships, indexes, migrations, or backup plan. The reel distinguishes table creation from database design.

## Verification Checklist
- Inspect schema normalization and relationships instead of accepting one-table growth.
- Check indexes for common query paths and performance-sensitive filters.
- Verify migrations, backup strategy, and query performance are part of the project.

## Skill Signal
Use this as a database design scan for schema shape, migration discipline, and query readiness.
