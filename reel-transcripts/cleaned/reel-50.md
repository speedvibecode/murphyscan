# Reel 50 - Agent Memory That Helps

## Summary
Agent memory should improve behavior without storing everything. The reel separates short-term conversation buffers from long-term persistent memory and stresses that only behavior-changing facts should be stored.

## Verification Checklist
- Keep recent session turns available and summarize older context when needed.
- Store durable preferences, decisions, and project context in a retrievable persistent layer.
- Test retrieval by asking the agent to use past context, and remove memories that do not affect behavior.

## Skill Signal
Use this as an agent-memory scan for retention, retrieval, and overcollection risk.
