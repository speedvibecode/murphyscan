# Reel 47 - Route AI Requests by Complexity

## Summary
Sending every AI task to the most expensive model is a billing failure. The reel recommends a complexity classifier, gateway-level routing, and recurring quality evaluations by model tier.

## Verification Checklist
- Classify requests by token count, task type, and reasoning depth.
- Route requests through one gateway endpoint that selects the cheapest adequate model.
- Evaluate model tiers weekly to confirm cost savings do not degrade quality.
