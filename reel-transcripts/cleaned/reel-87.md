# Reel 87 - Cloud Bills Reflect Architecture

## Summary
Unexpected cloud spend usually maps back to idle services, oversized instances, and unnoticed egress. The reel recommends auditing what is still running, sizing resources to real traffic, and tracing how data moves between regions and providers.

## Verification Checklist
- Audit idle environments, old replicas, buckets, and forgotten services that still incur cost.
- Compare actual utilization to provisioned size before assuming current capacity is justified.
- Trace cross-region and cross-provider traffic so egress costs do not stay hidden.

## Skill Signal
Use this as a cloud-cost scan for resource sprawl and data-movement overhead.
