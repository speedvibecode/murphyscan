# Reel 37 - Prepare for SOC 2

## Summary
Enterprise buyers eventually ask for security evidence. The reel recommends continuous compliance monitoring, automated evidence collection, and starting with SOC 2 Type 1 before working toward Type 2.

## Verification Checklist
- Connect infrastructure to a compliance monitoring tool or equivalent evidence process.
- Automate access review, vulnerability patching, backup proof, and restore evidence.
- Distinguish Type 1 readiness from Type 2 operating-history requirements.

## Skill Signal
Use this as an enterprise-readiness scan for evidence-backed compliance.
