# Reel 20 - CI/CD Safety Net

## Summary
Without version control and automated checks, recovery becomes guesswork. The reel frames CI/CD as the safety net that builds, tests, scans, deploys, and enables fast rollback.

## Verification Checklist
- Confirm the project uses git and production changes are traceable.
- Check CI for build, test, and vulnerability gates before deployment.
- Verify rollback to the last known-good version is fast and documented.

## Skill Signal
Use this as a CI/CD scan for automated gates and rollback capability.
