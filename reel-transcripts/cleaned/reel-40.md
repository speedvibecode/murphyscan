# Reel 40 - Mobile Deep Links

## Summary
Shared mobile links should open the app when installed and fall back to the web when not installed. The reel covers Apple Universal Links, Android App Links, and website fallback behavior.

## Verification Checklist
- Host the Apple app-site association file on the app domain.
- Host the Android asset links file and configure Android intent filters.
- Verify fallback behavior shows web content and an install prompt when the app is absent.

## Skill Signal
Use this as a mobile-linking scan for apps with shareable content.
