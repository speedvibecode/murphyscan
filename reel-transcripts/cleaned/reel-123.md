# Reel 123 - AI Code Needs Security Audits

## Summary
AI-generated code still needs production engineering discipline. The reel frames structured logging, alerting, runbooks, and security audits as the answer to common skepticism about vibe-coded systems. It reinforces that AI-generated code can ship only when operational and security checks are deliberate.

## Verification Checklist
- Verify generated code paths are covered by security audits or equivalent review.
- Check that structured logging, alerting, and runbooks exist for critical paths.
- Confirm AI-authored changes are not merged solely because they appear to work locally.
- Treat AI generation as a reason to increase evidence, not lower the bar.
