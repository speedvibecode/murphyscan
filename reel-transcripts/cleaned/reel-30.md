# Reel 30 - Start Charging with Stripe

## Summary
Builders often delay revenue by overbuilding billing. The reel recommends choosing a simple pricing model, using Stripe Checkout for payments, and using Stripe's hosted billing portal for post-sale account management.

## Verification Checklist
- Decide whether billing is flat, seat-based, usage-based, or hybrid.
- Use hosted checkout instead of custom card collection unless there is a strong reason.
- Provide a billing portal for plan changes, card updates, cancellations, and invoices.

## Skill Signal
Use this as a billing-readiness scan for apps that need to monetize.
