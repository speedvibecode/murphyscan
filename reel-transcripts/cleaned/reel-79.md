# Reel 79 - Pick the ORM for the Team You Have

## Summary
Prisma emphasizes schema-first guardrails and generated safety, while Drizzle stays closer to SQL and runtime control. The reel recommends matching the ORM to the team’s tolerance for abstraction, query ownership, and operational trade-offs like client weight and cold starts.

## Verification Checklist
- Check whether the team wants schema-driven guardrails or lighter direct query control.
- Evaluate runtime weight, cold-start impact, and migration workflow alongside type safety.
- Match the ORM choice to the engineering culture rather than treating it as a popularity contest.

Use this as an ORM-selection scan for abstraction fit and runtime trade-offs.
